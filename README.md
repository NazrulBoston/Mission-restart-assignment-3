# 📱 App Store 

## 📖 Description
App Store Clone is a React-based web application where users can browse different apps, view detailed information, and install them. Users can see app ratings, downloads, and reviews in the details page. When an app is installed, it is added to the Installation page and saved in localStorage so the installed apps remain even after refreshing the page. Users can also uninstall apps, which removes them from both the UI and localStorage. Toast notifications provide feedback when apps are installed or uninstalled.

## 🚀 Features
- Browse a list of apps
- View detailed app information
- Install apps
- Uninstall apps
- Installation list page
- Rating chart visualization
- Toast notifications
- Responsive design for mobile and desktop
- Data persistence using localStorage

## 🛠 Technologies Used
- React
- React Router
- Tailwind CSS
- DaisyUI
- React Icons
- Recharts (for rating charts)
- React Hot Toast
- Vite
